# 🐱 Meowventure

This is an old personal project I started around 6 years ago and never fully finished.
It is kept here as an archive of my early experiments with web development. It is very experimental and Spaghetti.

![Preview](preview.png)

## 📌 Status

⚠️ **Abandoned / Not actively maintained**

This project is no longer in development.  
It is preserved as-is for historical and educational purposes.

## 💡 About

Meowventure was an early attempt at building a browser-based game using React (Visual Novel style).

At the time, I was experimenting with:

- Game structure in the browser
- UI-driven gameplay ideas
- Component-based architecture

## 🛠️ Tech Stack

- React
- (Originally Create React App, migrated to Vite)
- JavaScript / JSX
- CSS

## 🚀 Running the project

If you still want to run it:

```bash
npm install
npm run dev
```
